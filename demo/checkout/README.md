# Checkout Flow Demo

A full-stack example application demonstrating a seamless e-commerce checkout experience. Note that this implementation uses mock payment processing; no real transactions occur.

## Tech Stack

- **Bun:** High-performance, all-in-one JavaScript runtime used as the full-stack server.
- **Frontend:** Built using React and Bun's native bundling capabilities.
- **Backend:** Logic and API routes handled within the Bun environment.
- **Storage:** All data kept in memory.

## Features

- **Cart Management:** Add, update, and remove items from the shopping cart.
- **Flexible Checkout:** Support for both guest checkout and authenticated user sessions.
- **Frictionless UX:** Streamlined flow for registered users with pre-filled details.

## Getting Started

Install dependencies:

```bash
bun install

```

Run the development server:

```bash
bun dev

```

Access the app: Open http://localhost:6002 in your browser.
