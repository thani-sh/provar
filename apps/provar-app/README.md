# Provar Editor App

A premium visual desktop editor for Provar test suites. Users can visually author, debug, and execute end-to-end tests with real-time feedback.

## Quick Start

1. Install dependencies from the monorepo root:
   ```bash
   bun install
   ```

2. Start the development server and Electrobun shell:
   ```bash
   bun --cwd apps/provar-app dev
   ```
